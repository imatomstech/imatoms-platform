🔬 Reliability Report
คำนวณ Reliability = e^(−Test time / MTTF)
🔒 Admin Only
📐 สูตรคำนวณ (Admin Only — Know How)
Reliability = exp( −Test_time / MTTF )
MTTF = [ Duty_Cycle(hr) − RepairTime(hr)/Count(times) ] × Load_Factor(%)
Duty_Cycle = Working_hrs_per_day × Working_days_per_year
Test_time = กำหนดเองตามการทดสอบ (ชั่วโมง)
Load_Factor = สัดส่วนโหลดจริง / โหลดสูงสุด (%)
